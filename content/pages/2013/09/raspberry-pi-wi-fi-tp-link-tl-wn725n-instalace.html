Raspberry Pi – Wi-Fi TP-LINK TL-WN725N instalace

12. 09. 2013 | Raspberry Pi

Aktuální návod najdete na https://raspberry.pavelrampas.cz/wi-fi/

Pro tuto wi-fi kartu je na Raspbian potřeba doinstalovat ovladač. Ten je možné případně stáhnout na jiném počítači a přenést si ho pomocí flash disku. Postupoval jsem podle návodu na tech.enekochan.com/2013/05/29/tp-link-tl-wn725n-version-2-in-raspberry-pi/, kde jsou případně další varianty. 

$ wget https://dl.dropboxusercontent.com/u/80256631/8188eu-20130209.tar.gz
$ tar -zxvf 8188eu-20130209.tar.gz
$ sudo install -p -m 644 8188eu.ko /lib/modules/`uname -r`/kernel/drivers/net/wireless
$ sudo depmod -a
$ sudo modprobe 8188eu

Aktualizace 26.12.2013
Ukázalo se že jakmile se změní jádro je třeba změnit i ovladač. Takže nejrůznější návody přestávají po krátké době fungovat. Nejideálnější je použít tento postup na foru https://forums.raspberrypi.com/viewtopic.php?p=462982. Stačí vybrat odpovídající soubor podle verze jádra a postupovat podle návodu. Verze se dá zjistit příkazem uname -r. V příkazech je třeba změnit název souboru. 

$ wget https://dl.dropboxusercontent.com/u/80256631/8188eu-2013xxyy.tar.gz
$ tar -zxvf 8188eu-2013xxyy.tar.gz
$ sudo cp rtl8188eufw.bin /lib/firmware/rtlwifi
$ sudo install -p -m 644 8188eu.ko /lib/modules/$(uname -r)/kernel/drivers/net/wireless
$ sudo insmod /lib/modules/$(uname -r)/kernel/drivers/net/wireless/8188eu.ko
$ sudo depmod -a
