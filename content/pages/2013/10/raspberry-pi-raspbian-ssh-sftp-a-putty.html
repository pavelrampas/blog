Raspberry Pi – Raspbian SSH, SFTP a PuTTY

25. 10. 2013 | Kategorie: Raspberry Pi

Nové články o Raspberry Pi najdete na https://raspberry.pavelrampas.cz/

Jak provozovat SSH a SFTP mezi Raspberry Pi s nainstalovaným Raspbianem a notebookem s Windows 7.

1. Napřed je potřeba obě zařízení spojit síťovým kabelem.

2. V Raspberry příkazem sudo raspi-config otevřeme konfiguraci a v Advance Options povolíme SSH, pak nastavíme statickou ip adresu. Příkazem sudo nano /etc/network/interfaces otevřeme soubor se síťovým nastavením.

Tuto řádku v souboru: 

iface eth0 inet dhcp

Nahradíme za tyto tři řádky: 

iface eth0 inet static
address 192.168.1.1
netmask 255.255.255.0

Nakonec Raspberry restartujeme.

3. Nastavení statické adresy ve Windows. Proklikáme se do nastavení sítě. Otevřeme Start/Ovládací panely/Centrum síťových připojení a sdílení, zde vybereme požadovanou síť. Klikneme na Vlastnosti, vybereme TCP/IPv4 a znovu klikneme na Vlastnosti. Tady zadáme ip adresu 192.168.1.2 a masku 255.255.255.0.

4. Když máme nastaveny ip adresy. Můžeme použít program PuTTY pro SSH komunikaci. Program stačí stáhnout a spustit, nemusí se instalovat. Po spuštění jenom zadáme ip adresu Raspberry a dáme připojit. Vyskočí nám okno ve kterém už se můžeme přihlásit na Raspberry pomocí přihlašovacího jména pi a hesla raspberry.

5. Pro přenos souborů pak můžeme použít SFTP. K tomu potřebujeme například program FileZilla ve kterém stačí vybrat nové SFTP připojení zadat ip adresu Raspberry přihlašovací jméno a heslo. Pak už stačí dát připojit.
